# NexFlow

Desktop companion app for the **XF-C5019D101** smart display. Customize a widget dashboard on your PC, then mirror it to the device with **Open Device**.

## Download

| Platform | File |
|----------|------|
| Windows | `NexFlow Setup 1.0.4.exe` |
| macOS | `NexFlow-1.0.4.dmg` |

## Requirements

- **Windows 10+** or **macOS 11+**
- XF-C5019D101 connected via USB

## Install

**Windows** — Run the `.exe` installer and launch NexFlow.

**macOS** — Open the `.dmg`, drag NexFlow to Applications, then launch.

On first run (macOS), grant **Accessibility** permission so window management works correctly.

## Quick Start

1. Connect the XF-C5019D101 via USB
2. Open NexFlow and arrange widget cards (clock, weather, music, to-do, etc.)
3. Click **Open Device** to display on the HUB
4. Click **Close Device** to stop and restore your windows
