# NexFlow

Desktop companion app for **Smartview Dock** (model **XF-C5019D101**). Customize a widget dashboard on your PC, then click **Start Live** to show it on the HUB display.

## Download

| Platform | File |
|----------|------|
| Windows | `NexFlow Setup x.x.x.exe` |
| macOS | `NexFlow-x.x.x.dmg` |

## Requirements

- **Windows 10+** or **macOS 11+**
- Smartview Dock (XF-C5019D101) connected via USB

## Install

**Windows** — Run the `.exe` installer and launch NexFlow.

**macOS** — Open the `.dmg`, drag NexFlow to Applications, then launch.

On first run (macOS), grant **Accessibility** permission so window management works correctly.

## Quick Start

1. Connect Smartview Dock via USB
2. Open NexFlow and arrange widget cards (clock, weather, music, to-do, etc.)
3. Click **Start Live** to display your layout on the HUB
4. Click **Stop Live** to stop and restore your windows
